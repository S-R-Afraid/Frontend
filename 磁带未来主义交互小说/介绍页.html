SYSTEM_CAPABILITIES // 系统能力

JSON 数据驱动

逻辑与代码完全分离。使用简单的 JSON 文件编写您的故事。支持本地文件热加载与自动读取。

富文本解析器

内置自定义语法解析：支持高亮颜色、下划线、删除线，以及可控的打字机停顿效果。

复古视觉核心

基于 CSS3 的纯代码视觉特效：模拟 80 年代 CRT 显示器的扫描线、电子束辉光及信号噪点。

可视化编辑器

包含一个独立的单文件节点编辑器。支持拖拽连线、实时图片预览及 JSON 导出。

DATA_PROTOCOL // 数据协议示例

请遵循以下协议定义您的故事节点：

{ "start": { "image": "images/city_ruins.jpg", "text": "系统初始化中... \\. \n欢迎来到 \\C[#0f0]{第七扇区}。", "options": [ { "text": "检查系统日志", "nextNode": "logs_entry" } ] } }

* 使用 \\. 插入 0.5 秒的打字停顿。
* 使用 \\C[color]{...} 改变文字颜色。
* 图片将自动应用 16-bit 像素化与 CRT 滤镜处理。
