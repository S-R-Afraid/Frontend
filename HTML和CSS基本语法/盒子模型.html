CSS 盒子模型（Box Model）演示

每一个 HTML 元素都可以看成一个盒子，由 content、padding、border、margin 组成。

content-box

width = 200px

padding = 20px

border = 5px

border-box

width = 200px

padding + border

都包含在 width 内

盒子模型计算方式说明

一、标准盒模型（content-box）

实际占用宽度 = content（200） + 左右 padding（20 × 2） + 左右 border（5 × 2） = 250px

二、怪异盒模型（border-box）

实际占用宽度 = width = 200px
content 会被自动压缩，以保证 padding 和 border 不会撑大盒子。

三、margin 的特点

margin 不属于盒子本身，用于控制元素与元素之间的距离， 在本示例中通过粉色背景区域进行可视化展示。

四、实际开发建议

  • 现代布局（尤其是响应式）推荐统一使用 box-sizing: border-box;
  • 可以通过 * { box-sizing: border-box; } 进行全局设置
